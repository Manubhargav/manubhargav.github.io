# manubhargav.github.io

Personal portfolio site for Manu Bhargav Srirangarajan — Fullstack Developer.  
🔗 **Live:** [manubhargav.github.io](https://manubhargav.github.io)

## Features

- **Dark themed UI** — Sleek dark interface with accent colors
- **Responsive layout** — Built with Bootstrap, works on all screen sizes
- **Project showcase** — Highlights personal projects with live links
- **Resume download** — One-click access to the latest resume
- **Contact section** — Direct email and text links
- **Social links** — GitHub, LinkedIn, Instagram, Twitter, Medium

## Tech Stack

- **Markup:** HTML5
- **Styling:** CSS3, Bootstrap 4
- **Scripts:** JavaScript, jQuery
- **Icons:** Font Awesome
